# RL Game

RL (Real Game) is a project that aims to create a minecraft-inspired game but more immersive. The game is designed to be lightweight and easy to run on a wide range of devices, while still providing an engaging and enjoyable gaming experience.
written in C++ using Vulkan for graphics rendering, the game features a voxel-based world that players can explore and interact with. The game includes a variety of biomes, creatures, and resources to discover.
